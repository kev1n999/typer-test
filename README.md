# ⌨️ Typer Test

**Typer Test** is a simple web automation project for **[Typing](https://www.typing.com/)**.

## Technologies Used
- Python
- Selenium
- PyAutoGUI
- PyFiglet
- CLI Box / Console Colors

## How to use
1. Clone the repository
```
git clone https://github.com/kev1n999/typer-test
```

3. Open the project root
```
cd typer-test
```

5. Create an virtual environment
```
python3 -m venv venv
source venv/bin/activate 
```

4. Install the dependencies
```
pip3 install -r requirements.txt
```

5. Run:
```
python3 main.py
```
